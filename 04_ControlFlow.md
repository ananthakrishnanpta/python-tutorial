
# Control Flow in Python

Control flow statements are used to determine the execution path of a program.

## Conditional Statements
### `if`, `elif`, `else`
```python
x = 10
if x > 0:
    print("Positive")
elif x < 0:
    print("Negative")
else:
    print("Zero")
```

## Loops
### `for` Loop
```python
for i in range(5):
    print(i)
```

### `while` Loop
```python
count = 0
while count < 3:
    print(count)
    count += 1
```
